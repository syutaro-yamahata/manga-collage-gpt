## 現在開発中

# 📚 漫画コラ画像生成アプリ（manga-collage-gpt）

画像をアップロードし、任意の文章で「どのように編集したいか」を指示すると、  
Flask サーバーが画像を加工し、コラ画像を生成するアプリです。

現在は Pillow を使った基本的な画像処理機能と、  
GPT による指示解析ロジックを搭載した軽量アプリとして構築しています。

---

## 🚀 機能一覧

- 画像アップロード（JPEG / PNG）
- テキスト指示に基づく画像編集処理
- 漫画コラ画像（吹き出し変更・表情変更など）生成の基盤
- Flask Web UI からの操作
- Pillow を使った画像加工
- GPT による編集指示解析（OpenAI API 連携予定）
